Gallery example

The code creates an image gallery with a button that allows adjusting the brightness of the displayed image. It uses JavaScript to handle events and modify the properties of elements on the page. When an image in the gallery is clicked, it is displayed in the main window. The "Darken" button changes its class, text, and also modifies the background transparency and opacity of the displayed image, creating a brightness adjustment effect.

IMG map example

This HTML code represents the markup for displaying an interactive map of planets using the 'img' and 'map' tags. The map shows images of planets and allows the user to click on each planet to access additional information through a hyperlink. 'area' tags are defined for the planets Venus, Earth, and Mars, with their respective alternative text and links. This code allows you to create an interactive map of planets where users can click on each planet to access additional information.
